Эта программма поможет следить за бюджетом на ваши покупки.

Чтобы испоьзовать эту программму вам нужен четкий бюджет на продукты и их список.

Программа принимает только положительные целые числа( > 0) и строки (строки можно вв одить только в список)

Если вам нужно завершить покупки или законччить список продуктов нужно вввести -1

АТРИБУТЫ:
--------
sum_of_buying :  int
    целое число
count : int
    целое число
price : int
    целое число
buying : boolean
    логическая переменная
start_sum : int
    целое число
list_of_products : list
    список
write_list : boolean
    логическаа переменная


МЕТОДЫ:
-------
itogi() --подводит итоги покупок

ПРИМЕЧАНИЕ:
-----------
Числа должны быть натуральными

ОШИБКИ:
-------
Если при запросе бюджета и цены ввести дробное число или символы отличные от цифр то вызовется ValueError"""

"""ИНФОРМАЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ (ИНСРУКЦИЯ ИСПОЛЬЗОВАНИЯ ПРОГРАММЫ)

Чтобы начать пользоваться программой вам нужно скачать любой Python IDE на ваше устройсво
(если он уже скачан то читайте дальше)

Для начала вас попросят ввести бюджет. Вы должны будете ввести НАТУРАЛЬНОЕ ЧИСЛО иначе вызовется ошибка

Далее вы должны ввести список покупок(Вводите название и нажимаете Enter).
Если вы закончили ввод в список покупок введите -1(если ввести -1 на первой иттерациий цикла то через 1 ввод вызовется ошибка)

Когда вы закончили со списком продуктов, то программа попросит ввести цену продукта
Вы должны будете ввести НАТУРАЛЬНОЕ ЧИСЛО иначе вызовется ошибка
Чтобы завершить процесс использования программы введите -1

После ввода цены ввыведется нынешний список покупок
Номер покупки ОБЯЗАТЕЛЬНО ВЫБИРАЙТЕ ИЗ УКАЗАНЫХ ИНАЧЕ ВЫВЕДЕТСЯ ОШИБКА
